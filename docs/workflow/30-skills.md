# Skills（渐进式披露）

技能目录：`.opencode/skills/<skill-name>/SKILL.md`

## 初始化相关

- `env-init`：首次初始化（uv + 前后端骨架 + 架构文档 + 可选模板启动）。
- `kickoff-rules`：启动治理协议（强制门禁/默认并行/落盘规则/opencode-only 约束）。
- `resume-handshake`：继续任务恢复协议。

## 门禁相关

- `code-audit`：审计式检查清单 + PASS/BLOCK。
- `test-coverage`：测试计划（本次改动+E2E+核心逻辑+每接口）。

## 复盘与复用

- `problem-finder`：先查历史再排查。

## UI/UX

- `ui-ux-pro-max`：可搜索的 UI/UX 数据集与栈指南。
