# Ant Design Pro 初始化

## 推荐初始化方式（pro-cli）

1. 安装 pro-cli（必须完成）：
   - `npm i @ant-design/pro-cli -g`
2. 创建项目：
   - `pro create myapp`
3. 选择 Umi 版本：
   - `umi@4`
4. 选择脚手架类型：
   - `simple`
5. 安装依赖（必须完成）：
   - `cd myapp && yarn`
   - 或 `cd myapp && npm install`
